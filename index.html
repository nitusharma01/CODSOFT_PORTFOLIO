Recipe Recommendation System

The Recipe Recommendation System is a web-based application that helps users discover healthy and suitable recipes based on their nutritional preferences and available ingredients. Users can input details such as Calories, Fat, Carbohydrates, Protein, Cholesterol, Sodium, Fiber, and Ingredients, and the system recommends recipes that best match their requirements. This project combines Machine Learning for intelligent recipe prediction with a Flask web framework to create a user-friendly interface. It provides an easy way for users to plan balanced meals and explore new dishes according to dietary needs.
